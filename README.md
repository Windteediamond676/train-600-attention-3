## 🖥️ What this app does

train-600-attention-3 is a small training app for a Toy GPT model.

It uses a short domain corpus and a three-context setup. That means it learns from a small block of text and tries to predict the next word from nearby words. It also uses attention, which helps the model focus on the most useful parts of the input.

This repo is built for learning and inspection. You can run it on Windows and watch how the model trains on a small text set.

## 📌 Key terms in plain English

- **Corpus**: the text the model reads
- **Token**: a small piece of text, like a word or part of a word
- **Next-token prediction**: guessing the next word or token
- **Attention**: a way for the model to focus on the most useful input
- **Context**: the nearby words the model uses to make a guess
- **Softmax**: a method that turns scores into chances

## 🔍 Why attention matters here

This repo includes attention because the model needs to decide which parts of the input matter most.

With a small three-context setup, the model only looks at a short window of text. Attention helps it weigh those words so the prediction can use the best signal in that small range.

That makes the training run easier to inspect and better for learning.
